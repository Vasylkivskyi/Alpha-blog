# Alpha blog

## Simple blog app created with Ruby on Rails

### Technologies were used:

- HTML
- Tailwind
- Ruby
- Ruby on Rails
- Javascript

## Installation

Clone the repo

Install the dependencies and gems

```sh
git clone git@github.com:Vasylkivskyi/Alpha-blog.git
```

```sh
cd Alpha-blog
```

```sh
bundle insall
```

Run migrations

```sh
rails db:migrate
```

Run the app

```sh
./bin/dev
```
